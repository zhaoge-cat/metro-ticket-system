# 🚇 地铁查询售票系统

基于 Python + SQLite + Streamlit 开发的地铁查询售票系统，支持线路查询、站点查询、路径规划、购票及数据管理等功能。

## ✨ 功能特性

- **线路查询** - 查看所有地铁线路信息
- **站点查询** - 支持站点名称模糊搜索
- **路径规划** - 同线路直达/跨线路换乘查询
- **购票功能** - 票价计算、多人购票、历史记录
- **数据管理** - 添加/删除/修改线路和站点
- **统计信息** - 票房和售票数据统计

## 🛠️ 技术栈

- **Python 3.10+** - 开发语言
- **SQLite 3** - 数据库
- **Streamlit** - 网页界面框架
- **Pandas** - 数据处理
- **Matplotlib** - 图表展示

## 📁 项目结构

```
├── metro_streamlit.py    # Streamlit 网页版主程序
```

## 🚀 快速开始

### 安装依赖

```bash
pip install streamlit pandas matplotlib
```

### 运行网页版

```bash
streamlit run metro_streamlit.py
```

### 运行命令行版

```bash
python metro_cli.py
```

## 📊 数据库结构

系统包含 6 张数据表：

| 表名 | 说明 |
|------|------|
| `metro_lines` | 线路信息（线路ID、名称、颜色） |
| `stations` | 站点信息（站点名称、所属线路、位置） |
| `station_connections` | 站点连接关系 |
| `pricing` | 票价规则（阶梯计价） |
| `tickets` | 车票记录（购票历史） |
| `station_traffic` | 站点流量统计 |

## 🎯 使用说明

### 1. 线路查询
- 查看所有线路列表
- 点击线路查看详细站点信息

### 2. 站点查询
- 输入站点名称进行模糊搜索
- 支持拼音和中文搜索

### 3. 路径规划
- 选择起点站和终点站
- 系统自动计算最优路径和票价
- 支持同线路和跨线路换乘

### 4. 购票功能
- 选择起点和终点
- 系统自动计算票价
- 支持多人购票

### 5. 数据管理
- 添加新线路
- 添加/删除/修改站点
